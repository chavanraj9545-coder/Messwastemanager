 🍽️ Zero Waste Mess Management System

A full-stack smart mess management platform designed to reduce food wastage using attendance tracking, inventory management, analytics, and machine learning-based demand prediction.

## 🚀 Project Overview

The Zero Waste Mess Management System helps educational institutions and hostels efficiently manage food preparation and consumption while minimizing waste.

The system combines modern web technologies and machine learning to forecast food demand and improve resource utilization.

## ⚙️ Installation

Clone Repository

```bash
git clone <repository-url>
```

Install Frontend Dependencies

```bash
npm install
```

Install Backend Dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
uvicorn main:app --reload
```

Run Frontend

```bash
npm run dev
```
